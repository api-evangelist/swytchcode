# Swytchcode

An API execution layer / runtime for AI agents and developers, delivered primarily as a CLI (npm-distributed) with a local/self-hosted MCP server. It sits between agents and 2,000+ production APIs, handling auth injection, retries, idempotency, schema validation, and policy control.
